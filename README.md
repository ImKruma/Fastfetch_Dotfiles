## INSTALLATION
```
git clone https://github.com/ImKruma/Fastfetch_Dotfiles.git ~/.config/fastfetch
```
## You may need to install the JetBrainsMono Nerd Font as well if you dont have it
```
sudo pacman -S ttf-jetbrains-mono-nerd
```
